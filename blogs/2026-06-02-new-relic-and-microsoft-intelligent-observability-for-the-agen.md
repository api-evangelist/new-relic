New Relic and Microsoft have developed deep integrations to embed observability into Azure workflows, centering on the New Relic MCP Server which makes observability data accessible to AI agents through the Model Context Protocol for automated incident detection and remediation. Key features include Azure SRE Agent capabilities, New Relic Monitoring for Microsoft Foundry, and SAP monitoring solutions available through Microsoft Marketplace. The partnership enables engineering teams to manage infrastructure and AI applications without manually bridging between systems.
